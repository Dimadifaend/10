
# Домашнее задание к работе 10

## Условие задачи
Составьте программу, которая для заданной пользователем фигуры, например прямоугольника (треугольника или другой фигуры см. инидивидуальное задание работы 9) предлагает меню выбора одной из операций:
1) рассчитать площадь;
2) вывести определение фигуры;
3) нарисовать фигуру

## 1. Алгоритм и блок-схема

### Алгоритм

##### 1 Начало

2 Объявить переменные для ввода пользователя:

      cet - выбор операции

3 Вывести меню выбора операций

4 Ввести значение cet от пользователя

5 Проверить значение cet:

  
      Если cet = 1: перейти к расчету площади
      Если cet = 2: перейти к выводу определения  
      Если cet = 3: перейти к рисованию фигуры
      Иначе: вывести сообщение об ошибке

       Ввести высоту h, основания a и b

       Вывести результат

      Вывести текст определения равнобедренной трапеци
